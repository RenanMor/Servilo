# Servilo
Projeto para disciplina ES44C - Programação Web Front-End - ES41 (2025_02)

## Descrição do Projeto
Este projeto consiste na criação de um website simples, denominado "Mural de Serviços do Bairro", com o objetivo de conectar moradores e prestadores de serviços locais. A plataforma visa resolver a dificuldade dos moradores em encontrar profissionais confiáveis e, ao mesmo tempo, auxiliar os profissionais locais a divulgarem seus trabalhos para a comunidade.

### Problema
Moradores de um bairro têm dificuldade em encontrar e contratar prestadores de serviços locais confiáveis (como encanadores, eletricistas, diaristas, babás) e, ao mesmo tempo, esses profissionais locais têm dificuldade em divulgar seu trabalho para os vizinhos.

### Solução
Um "Mural de Serviços do Bairro" onde profissionais locais podem se cadastrar com seu serviço, contato e área de atuação (limitada ao bairro ou região). Os moradores podem buscar pelo serviço que precisam e ver avaliações deixadas por outros vizinhos, criando uma rede de confiança e fomentando a economia local. A cidade foco é Cornélio Procópio, Paraná, Brasil.

### Objetivos de Desenvolvimento Sustentável (ODS)
O projeto contribui diretamente para:
- **ODS 8: Trabalho Decente e Crescimento Econômico**: Promovendo o emprego produtivo e o trabalho decente para todos.
- **ODS 11: Cidades e Comunidades Sustentáveis**: Fortalecendo os laços comunitários e a economia local.

## Estrutura do Projeto
O projeto é composto por diversas páginas HTML principais e um arquivo CSS para estilização:
- `index.html`: Página principal do website, apresentando a proposta da solução, o problema que resolve e os ODS atendidos.
- `cadastro.html`: Página de cadastro para novos prestadores de serviços, contendo um formulário com campos para nome, e-mail, telefone, serviço oferecido, área de atuação e senha. Após o cadastro, redireciona para a página de perfil.
- `perfil.html`: Página de perfil do usuário, exibindo foto, serviço oferecido, localização e um link para o currículo completo do profissional.
- `mural.html`: Página de mural de serviços, onde são listados exemplos de trabalhadores cadastrados com foto, breve descrição, avaliação e um botão para contratar o serviço.
- `style.css`: Arquivo de estilos CSS para todas as páginas do projeto.
